# Theory of Computation (TOC) - C++ Codes

This repository contains C++ implementations of various algorithms and concepts related to the Theory of Computation (TOC).

## Structure

- `src/`: Contains all the C++ source code files.

## Contents

The codes cover topics such as:

- Finite Automata
- Turing Machines
- And other TOC concepts.

## How to Run

You can compile any of the files using a C++ compiler (like g++).

Example:

```bash
g++ src/tocfile12.cpp -o tocfile12
./tocfile12
```
